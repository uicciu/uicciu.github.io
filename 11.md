# (1+1) SIMPLE CNN CRYPTANALYTIC EXTRACTION PROJECT

## Project Overview
实验首次首次恢复了采用最大池化层的卷积神经网络（CNN）的参数，本项目展示了黑盒情况下，成功恢复由随机数据集训练的两层(1+1)CNN模型权重的完整过程。

## 重要文件简述
1. **common.py**:提供基础配置以及模型定义，存储模型信息用于准确度测试。
2. **extract_signature_cuda.py**:利用GPU加速提取CNN模型卷积层签名，利用奇异值分解（SVD）处理从池化层变化点(PSP)收集到的差异向量，求解齐次线性方程组，提取模型权重。
3. **reconstructCNN.py**:包含重构模型参数以及误差分析模块，利用提取的向量恢复完整卷积核权重矩阵，并与真实值进行精确度比较。
